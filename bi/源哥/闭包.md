# 私有变量

```javascript
function createCounter() {
  var count = 0;
  return {
    increment: function() {
      count++;
      return count;
    },
    reset: function() {
      count = 0;
      return count;
    }
  };
}
var counter = createCounter();
console.log(counter.increment()); // ?
console.log(counter.reset()); // ?
console.log(counter.increment()); // ?
```

**代码**：https://code.juejin.cn/pen/7522716656838115366

```plain text
输出 1、0、1。闭包 increment 和 reset 共享并修改 createCounter 作用域中的 count
```

> 问题：什么时候用过闭包？
> 
> 类似于图中这种例子，需要封装，使用闭包的特性，让内部的参数变成**私有变量**

# 4. 内存

> 以下代码是否存在**内存溢出**的风险？

```javascript
function outer() {
  var largeData = new Array(1000000).fill(0); // 大数组
  return function inner() {
    console.log(largeData.length);
  };
}
var closure = outer();
closure(); // 多次调用
```

```plain text
存在风险。inner 闭包引用 largeData，导致其无法被垃圾回收。
优化方法：
在闭包不再使用时将其设为 null
使用完closure 后，设置为null
```

> 真实场景中，此处会描述为正常且有必要保留的参数，只是会比较注意占的比较大的内存，是否有必要

# 原型链

```javascript
const module = (function() {
  let privateVar = 'secret'; // 闭包中的私有变量
  
  function PrivateMethod() {
    this.privateVar = 'overridden'; // 实例属性
  }

  PrivateMethod.prototype.getPrivateVar = function() {
    return privateVar; // 通过原型方法访问闭包变量
  };

  PrivateMethod.prototype.setPrivateVar = function(value) {
    privateVar = value; // 通过原型方法修改闭包变量
  };

  return {
    createInstance: function() {
      return new PrivateMethod();
    }
  };
})();

// 执行代码
const instance1 = module.createInstance();
const instance2 = module.createInstance();

console.log(instance1.getPrivateVar()); // ？
console.log(instance1.privateVar);       // ？

instance1.setPrivateVar('new secret');
console.log(instance2.getPrivateVar()); // ？
console.log(privateVar);                // ？
```

**代码**：https://code.juejin.cn/pen/7522717115959541787

```plain text
输出 1：secret
原型方法 getPrivateVar 访问闭包中的 privateVar，未被实例属性覆盖。

输出 2：overridden
实例属性 privateVar 直接赋值，优先级高于原型链。

输出 3：new secret
setPrivateVar 修改闭包中的 privateVar，所有实例共享该变量。

输出 4：报错（privateVar 未定义）
闭包外无法直接访问 privateVar，外部作用域无该变量。
```

## demo4 

> 编写一个模块，使用闭包实现以下功能：
> 
> - 私有变量 `cache` 存储数据。
> 
> - 公共方法 `set(key, value)` 存入数据。
> 
> - 公共方法 `get(key)` 获取数据。

**代码** https://code.juejin.cn/pen/7522721450607411251

**提示：**参考**私有变量**

```javascript
const cacheModule = (function() {
  var cache = {};
  return {
    set: function(key, value) {
      cache[key] = value;
    },
    get: function(key) {
      return cache[key];
    }
  };
})();
cacheModule.set('name', 'Alice');
console.log(cacheModule.get('name')); // 'Alice'
```

## demo5

> 编写一个递归函数计算斐波那契数列，使用闭包缓存结果以提升性能。

**代码** https://code.juejin.cn/pen/7522721642439508014

```javascript
function fibonacci() {
  var cache = {};
  return function(n) {
    if (n in cache) return cache[n];
    if (n <= 1) return n;
    cache[n] = arguments.callee(n-1) + arguments.callee(n-2);
    return cache[n];
  };
}
const fib = fibonacci();
console.log(fib(10)); // 55
```
