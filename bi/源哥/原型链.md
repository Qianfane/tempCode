# 原型链查找

## demo1

```javascript
function Foo() {
  this.bar = 'bar';
}
Foo.prototype.baz = 'baz';

const foo = new Foo();

console.log(foo.bar); // ?
console.log(foo.baz); // ?
console.log(foo.qux); // ?
```

**代码** https://code.juejin.cn/pen/7523049273379094574

```html
foo.bar：'bar'（属性存在于实例本身）
foo.baz：'baz'（实例本身无baz，沿原型链找到Foo.prototype）
foo.qux：undefined（原型链上无此属性，查找失败）
```

# 继承

## 如何实现原型链？

**代码** https://code.juejin.cn/pen/7523051010021916718

```javascript
function Parent() { this.name = 'parent'; }
Parent.prototype.getName = function() { return this.name; };

function Child() {}
// 原型链继承：子类原型指向父类实例
Child.prototype = new Parent();
Child.prototype.constructor = Child; // 修复constructor指向

const c = new Child();
console.log(c.getName()); // 'parent'
```

## 如何实现组合继承？

**代码** https://code.juejin.cn/pen/7523051335554760731

```javascript
function Parent(name) { this.name = name; }
Parent.prototype.getName = function() { return this.name; };

function Child(name, age) {
  Parent.call(this, name); // 构造函数继承：继承属性，避免引用类型共享
  this.age = age;
}
// 原型链继承：继承方法
Child.prototype = new Parent();
Child.prototype.constructor = Child; // 修复constructor
Child.prototype.getAge = function() { return this.age; };

const c = new Child('张三', 18);
console.log(c.getName()); // '张三'
console.log(c.getAge()); // 18
```

# 函数 vs 方法 vs 构造函数 vs 箭头函数

```javascript
函数：独立的代码块，能够接受参数并返回值
方法：依附于对象或者类的函数，通过对象的属性来进行调用
构造函数：专门用于创建和初始化对象，会跟new搭配
箭头函数：简洁的函数表达式
```

# 手写题

## demo1

手写myNew

```sql
function myNew(constructor, ...args) {
  // 1. 创建空对象
  const obj = {};
  // 2. 绑定原型链
  obj.__proto__ = constructor.prototype;
  // 3. 执行构造函数
  const result = constructor.apply(obj, args);
  // 4. 返回结果
  return typeof result === 'object' && result !== null ? result : obj;
}
```
