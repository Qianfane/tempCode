# box-sizing 与背景裁剪

```css
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid #000;
  box-sizing: border-box;
  background-color: orange;
  background-clip: content-box;
}
```

代码：https://code.juejin.cn/pen/7523040371920764979

```css

box-sizing: border-box 对 width 的影响
background-clip 与盒模型的关系
```

# 4. input 与 button 的盒模型差异

解释下以下代码宽度的差异

```html
<div class="container">
  <input type="text" class="elem" value="Input">
  <button class="elem">Button</button>
</div>

<style>
.elem {
  width: 100%;
  padding: 10px;
  border: 2px solid royalblue;
}
</style>
```

**代码**：https://code.juejin.cn/pen/7523041642559995913

```html
input 默认使用标准盒模型（box-sizing: content-box），总宽度 = width + 2*padding + 2*border
button 默认使用怪异盒模型（box-sizing: border-box），总宽度 = width（已包含 padding 和 border）

需要统一处理box-sizing
```

# 5. 外边距合并与塌陷

分析以下代码的问题,并提供解决方案

```html
<div class="parent">
  <div class="child">内容</div>
</div>

<style>
    .parent {
      margin-top: 50px;
      background: #f5f5f5;
    }
    .child {
      margin-top: 30px;
      padding: 20px;
    }
</style>
```

**代码**：https://code.juejin.cn/pen/7523044079379152923

```html
父元素与子元素的 margin-top 合并，导致父元素实际顶部外边距为 50px（取较大值）

解决方案
- 给父元素添加 padding-top: 1px 或 border-top: 1px solid transparent。
- 设置父元素 overflow: hidden。
- 使用 Flex 布局：display: flex。
```

# 浮动计算

计算浮动元素的总宽度，并解释其布局行为。

```html
.float-box {
  float: left;
  width: 200px;
  padding: 10px;
  border: 3px solid #333;
  margin: 5px;
}
```

**代码** https://code.juejin.cn/pen/7523045212009005106

```html
总宽度：200px + 2*10px + 2*3px + 2*5px = 236px（标准盒模型）。

布局行为：
浮动元素脱离文档流，宽度默认由内容撑开（类似 inline-block）。
margin 不会与父元素或兄弟元素合并。
```

# 场景题

## demo1

实现一个响应式卡片组件，要求：

- 卡片宽度为父容器的 30%，间距 5%。

- 卡片内部有 10px 内边距，边框 2px。

- 在手机端（<600px）时，卡片宽度变为 90%，且边框颜色变为红色。

**代码 ** https://code.juejin.cn/pen/7523047126621700134

```yaml
.card {
  width: 30%;
  margin-right: 5%;
  padding: 10px;
  border: 2px solid #ccc;
  box-sizing: border-box;
  float: left;
}
.card:last-child { margin-right: 0; }

/* 响应式调整 */
@media (max-width: 600px) {
  .card {
    width: 90%;
    margin-right: 5%;
    border-color: red;
    float: none; /* 取消浮动以适应移动端单列布局 */
  }
}
```

## demo2

使用负外边距实现元素重叠效果，要求：

- 两个相邻元素的水平间距为 - 20px（即重叠 20px）。

- 元素本身的宽度为 200px，内边距 10px，边框 1px。

**代码** https://code.juejin.cn/pen/7523048192586776627

```yaml
.element {
  width: 200px;
  padding: 10px;
  border: 1px solid #000;
  box-sizing: border-box;
  margin-right: -20px; /* 负外边距实现重叠 */
  float: left;
}
```

> Margin 可为负数【重叠】
