# 前端水印

自定义指令：前端水印



水字数的工具类

目前web前端常做的是明水印，实现基本上是第三方，聊以下话题

- **前端浏览器加水印**：

    - 减轻服务端的压力，快速反应

    - 安全系数较低，对于掌握一定前端知识的人来说可以通过各种骚操作跳过水印获取到源文件

    - 适用场景：
    资源不跟某一个单独的用户绑定，而是一份资源，多个用户查看，需要在每一个用户查看的时候添加用户特有的水印，多用于某些机密文档或者展示机密信息的页面，水印的目的在于文档外流的时候可以追究到责任人

- **后端服务器加水印**：

    - 当遇到大文件密集水印，或是复杂水印，占用服务器内存、运算量，请求时间过长

    - 安全性高，无法获取到加水印前的源文件

    - 适用场景：资源为某个用户独有，一份原始资源只需要做一次处理，将其存储之后就无需再次处理，水印的目的在于标示资源的归属人

    

前端⽔印可以分为**明⽔印**和**暗⽔印**两种类型。

1. **明⽔印**：明⽔印是通过在⽂本或图像上覆盖另⼀层图像或⽂字来实现的。这种⽔印会明显地出现在⻚⾯上，可以⽤来显⽰版权信息或其他相关信息。 

2. **暗⽔印**：暗⽔印是指在⽂本或图像中隐藏相关信息的⼀种技术。这种⽔印不会直接出现在⻚⾯上，只有在特殊的程序或⼯具下才能被检测到。暗⽔印通常⽤于保护敏感信息以及追踪⽹⻚内容的来源和版本。 



**实现**：

1. 重复的dom元素覆盖实现： 在页面上覆盖一个position:fixed的div盒子，盒子透明度设置较低，设置pointer\-events: none;样式实现点击穿透，在这个盒子内通过js循环生成小的水印div，每个水印div内展示一个要显示的水印内容

2. canvas输出背景图： 绘制出一个水印区域，将这个水印通过toDataURL方法输出为一个图片，将这个图片设置为盒子的背景图，通过backgroud\-repeat：repeat；样式实现填满整个屏幕的效果。

3. svg实现背景图： 与canvas生成背景图的方法类似，只不过是生成背景图的方法换成了通过svg生成

## 暗水印

用于在数字图像或音频中嵌入版权信息，以防止未经授权的使用和保护知识产权，如鲁棒盲水印，包含以下特性

- **鲁棒性**：鲁棒盲水印能够在面对多种攻击时保持水印的完整性，如图像的旋转、平移、缩放、裁剪、加噪以及音频的重采样、格式转换等，都能有效抵御这些操作对水印的破坏。

- **盲性**：盲水印技术允许在没有原始载体的情况下检测和提取水印，这使得版权持有者无需保留原始文件即可验证内容的版权归属。

**具体实现方法**

在数字图像中，鲁棒盲水印通常通过以下步骤实现：

1. **频域变换**：将图像从空域转换到频域，利用傅里叶变换等技术。

2. **水印嵌入**：在频域的高频部分嵌入水印信息，因为高频成分通常对应于图像的细节，对视觉影响较小，同时水印信息分散在全图，不易被破坏。

3. **逆变换**：将修改后的频域图像转换回空域，得到加水印后的图像。

在音频中，鲁棒盲水印可能采用的技术包括：

- **扩频技术**：通过扩散水印信号，使其不易被察觉且难以去除。

- **量化调制**：利用音频信号的量化误差来嵌入水印。

- **最低有效位方法**：在音频数据的最低有效位嵌入水印，影响较小但鲁棒性较差。

- **经验模式分解**：将音频信号分解成固有模态分量，再在这些分量上嵌入水印。

**鲁棒盲水印的挑战**

- **隐匿性和鲁棒性的权衡**：提高鲁棒性往往需要增加水印的强度，这可能导致对原始数据的可见干扰。

- **攻击类型多样化**：不同类型的攻击（如JPEG压缩、噪声添加、重采样等）对水印的影响不同，需要设计鲁棒性较强的算法来应对。

- **盲提取的限制**：虽然盲水印可以无需原始载体提取，但其提取的准确性和稳定性可能受攻击类型和强度的影响。



# 常见实现

```JavaScript
// directives/watermark.js
import { ref, watchEffect, onMounted, onUnmounted } from 'vue';

export const vWatermark = {
  mounted(el, binding) {
    const options = parseOptions(binding);
    const watermarkId = `watermark-${Date.now()}-${Math.random().toString(36).substring(2, 8)}`;
    
    // 创建水印容器
    const watermarkContainer = createWatermarkContainer(watermarkId, options);
    el.appendChild(watermarkContainer);
    
    // 响应式更新
    const resizeObserver = new ResizeObserver(() => {
      updateWatermarkPosition(watermarkContainer, el, options);
    });
    resizeObserver.observe(el);
    
    // 存储引用以便后续清理
    el._watermark = {
      container: watermarkContainer,
      observer: resizeObserver
    };
  },
  
  updated(el, binding) {
    const options = parseOptions(binding);
    const { container } = el._watermark;
    
    // 更新水印样式
    updateWatermarkStyle(container, options);
    updateWatermarkPosition(container, el, options);
  },
  
  unmounted(el) {
    const { container, observer } = el._watermark;
    if (observer) observer.disconnect();
    if (container && container.parentNode === el) {
      el.removeChild(container);
    }
    delete el._watermark;
  }
};

// 解析指令参数
function parseOptions(binding) {
  const value = binding.value || {};
  return {
    content: value.content || '水印内容',
    fontSize: value.fontSize || 16,
    fontColor: value.fontColor || 'rgba(0,0,0,0.1)',
    rotate: value.rotate || -15,
    gapX: value.gapX || 100,
    gapY: value.gapY || 150,
    zIndex: value.zIndex || 1000,
    width: value.width || 200,
    height: value.height || 100,
    // 响应式配置
    responsive: value.responsive !== false,
    refreshRate: value.refreshRate || 300
  };
}

// 创建水印容器
function createWatermarkContainer(id, options) {
  const container = document.createElement('div');
  container.id = id;
  container.className = 'watermark-container';
  container.style.pointerEvents = 'none';
  container.style.position = 'absolute';
  container.style.inset = '0';
  container.style.overflow = 'hidden';
  container.style.zIndex = options.zIndex;
  
  // 创建 SVG 水印图案
  const svg = createWatermarkSvg(options);
  container.appendChild(svg);
  
  return container;
}

// 创建水印 SVG
function createWatermarkSvg(options) {
  const svg = document.createElementNS('http://www.w3.org/2000/svg', 'svg');
  svg.setAttribute('width', '100%');
  svg.setAttribute('height', '100%');
  svg.setAttribute('xmlns', 'http://www.w3.org/2000/svg');
  
  const defs = document.createElementNS('http://www.w3.org/2000/svg', 'defs');
  svg.appendChild(defs);
  
  // 创建水印图案
  const pattern = document.createElementNS('http://www.w3.org/2000/svg', 'pattern');
  pattern.setAttribute('id', 'watermarkPattern');
  pattern.setAttribute('width', options.gapX);
  pattern.setAttribute('height', options.gapY);
  pattern.setAttribute('patternUnits', 'userSpaceOnUse');
  defs.appendChild(pattern);
  
  // 创建文本元素
  const text = document.createElementNS('http://www.w3.org/2000/svg', 'text');
  text.setAttribute('x', options.width / 2);
  text.setAttribute('y', options.height / 2);
  text.setAttribute('font-size', options.fontSize);
  text.setAttribute('fill', options.fontColor);
  text.setAttribute('text-anchor', 'middle');
  text.setAttribute('dominant-baseline', 'middle');
  text.setAttribute('transform', `rotate(${options.rotate} ${options.width / 2} ${options.height / 2})`);
  text.textContent = options.content;
  pattern.appendChild(text);
  
  // 创建背景
  const rect = document.createElementNS('http://www.w3.org/2000/svg', 'rect');
  rect.setAttribute('width', '100%');
  rect.setAttribute('height', '100%');
  rect.setAttribute('fill', 'url(#watermarkPattern)');
  svg.appendChild(rect);
  
  return svg;
}

// 更新水印位置（处理滚动和缩放）
function updateWatermarkPosition(container, parent, options) {
  if (!container || !parent) return;
  
  // 处理滚动容器
  if (parent.scrollWidth > parent.clientWidth || parent.scrollHeight > parent.clientHeight) {
    container.style.position = 'relative';
    container.style.width = '100%';
    container.style.height = '100%';
  } else {
    container.style.position = 'absolute';
  }
}

// 更新水印样式
function updateWatermarkStyle(container, options) {
  const svg = container.querySelector('svg');
  const pattern = svg?.querySelector('#watermarkPattern');
  const text = pattern?.querySelector('text');
  
  if (text) {
    text.setAttribute('font-size', options.fontSize);
    text.setAttribute('fill', options.fontColor);
    text.setAttribute('transform', `rotate(${options.rotate} ${options.width / 2} ${options.height / 2})`);
    text.textContent = options.content;
  }
  
  if (pattern) {
    pattern.setAttribute('width', options.gapX);
    pattern.setAttribute('height', options.gapY);
  }
}  
```
